Event loop function to handle incoming connections and data. 

This function waits for events such as new connections or incoming data. It handles each event accordingly\+:
\begin{DoxyItemize}
\item Accepts new client connections.
\item Reads data from connected clients and sends the length of the received data back.
\item Closes connections in case of errors or if no data is received.
\end{DoxyItemize}
