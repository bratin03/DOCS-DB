\doxysubsection{Detailed Description}
A structure to represent a key-\/value pair. 

This structure holds a {\ttfamily key} and a {\ttfamily val} as strings. It provides utility functions to check if the pair is empty, convert the pair to a log entry string, and split a log entry string back into a key-\/value pair.
