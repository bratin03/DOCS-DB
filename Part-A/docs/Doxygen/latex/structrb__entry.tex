\doxysubsection{Detailed Description}
Represents an entry in a Red-\/\+Black tree, used in the Memtable or Sparse Index. 

This structure contains a {\ttfamily key} of type {\ttfamily std\+::string} and a {\ttfamily val} which can be either a string (for the Memtable) or an {\ttfamily uint64\+\_\+t} (for the Sparse Index, representing an offset inside the S\+ST). It also supports operations like calculating the size of the entry and comparison based on the key.

Calculates the size of the entry. 

This function returns the size of the entry, which is the size of the key plus the size of the value.
\begin{DoxyItemize}
\item If the value is empty ({\ttfamily std\+::nullopt}), it returns the size of the key.
\item If the value is a string, it adds the size of the string.
\item If the value is an {\ttfamily uint64\+\_\+t}, it adds 8 bytes (the size of a 64-\/bit integer).
\end{DoxyItemize}
