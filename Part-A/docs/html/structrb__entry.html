Detailed Description

Represents an entry in a Red-Black tree, used in the Memtable or Sparse Index.

This structure contains a key of type std::string and a val which can be either a string (for the Memtable) or an uint64_t (for the Sparse Index, representing an offset inside the SST). It also supports operations like calculating the size of the entry and comparison based on the key.

Compares two rb_entry objects based on the key.

This function compares the keys of two Red-Black tree entries and returns the comparison result. It uses the <=> (three-way comparison) operator and returns:

  • std::strong_ordering::equal if the keys are the same.
  • std::strong_ordering::less if this entry's key is smaller.
  • std::strong_ordering::greater if this entry's key is larger.

uint64_t rb_entry::size ( ) const
inline

Calculates the size of the entry.

This function returns the size of the entry, which is the size of the key plus the size of the value.

  • If the value is empty (std::nullopt), it returns the size of the key.
  • If the value is a string, it adds the size of the string.
  • If the value is an uint64_t, it adds 8 bytes (the size of a 64-bit integer).
Returns
The size of the entry in bytes.
