Simulates a client sending multiple packets to a server and measuring the round-trip latency.

This function connects to a server, sends a series of packets, and measures the round-trip time (latency) for each packet. The client sends packets of random sizes between 512 and 1024 bytes and logs the latency and bandwidth metrics in a log file.
